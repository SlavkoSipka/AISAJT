───────────────────────────────────────────────────────────────────────────────
⭐⭐⭐ PRIORITET 1 - LEAD GENERATION (NAJBITNIJI!) ⭐⭐⭐
───────────────────────────────────────────────────────────────────────────────

Event Name: generate_lead

📍 Gde se okida:
   - Kada korisnik uspešno pošalje kontakt formu i stigne na Thank You page (/thank-you)
   - Thank You page se otvara automatski nakon uspešnog slanja forme sa Contact page ili Home page

📊 Parametri:
   - event_category: "Lead Generation"
   - event_label: "Contact Form - Success" ili "Home Contact Form - Success"
   - value: 1
   - currency: "EUR"
   - lead_source: "contact_page" ili "home_page"
   - language: "sr" ili "en"
   - user_name: Ime korisnika

🎯 Kako dodati u GA4:
   1. Idite na Google Analytics → Configure → Events
   2. Pronađite event: "generate_lead"
   3. Kliknite "Mark as conversion" ✅
   4. Sačuvajte
   
   ⭐ OVO JE NAJVAŽNIJI EVENT - praćenje lead-ova! ⭐


───────────────────────────────────────────────────────────────────────────────
⭐⭐ PRIORITET 2 - CTA BUTTONS (Dugmad koja vode na Contact) ⭐⭐
───────────────────────────────────────────────────────────────────────────────

Event Name: cta_click

📍 Gde se okida:
   1. "Kontakt" dugme u Top Navigation
   2. "Kontakt" dugme u Mobile Menu
   3. "Kontakt" dugme u Side Navigation
   4. "Zakažite Besplatnu Konsultaciju" - Video sekcija
   5. "Saznaj Više - Web Dizajn" - Services sekcija
   6. "Saznaj Više - Baze Podataka" - Services sekcija
   7. "Saznaj Više - Online Marketing" - Services sekcija
   8. "Razgovarajmo o Vašem Projektu" - Portfolio sekcija

📊 Parametri:
   - event_category: "Engagement"
   - event_label: Naziv dugmeta (npr. "Kontakt - Top Navigation")
   - location: Lokacija na sajtu (npr. "top_nav", "services_section")
   - language: "sr" ili "en"

🎯 Kako dodati u GA4:
   1. Idite na Reports → Engagement → Events
   2. Pronađite event: "cta_click"
   3. Kreirajte custom report:
      - Dimension: event_label
      - Metric: Event count
   4. Videćete koji CTA dugmad najviše klikaju korisnici

KAKO PRATITI EVENTOVE U GOOGLE ANALYTICS 4
═══════════════════════════════════════════════════════════════════════════════

───────────────────────────────────────────────────────────────────────────────
KORAK 1: Real-Time Provera (Da li eventi stižu?)
───────────────────────────────────────────────────────────────────────────────

1. Idite na: https://analytics.google.com
2. Izaberite Property: G-6C046QS9HG
3. Kliknite: Reports → Realtime
4. Testirajte akciju na sajtu (npr. pošaljite formu)
5. Za 5-30 sekundi videćete event u "Event count by Event name"


───────────────────────────────────────────────────────────────────────────────
KORAK 2: Kreiranje Konverzija (Za lead tracking)
───────────────────────────────────────────────────────────────────────────────

1. Idite na: Configure → Events
2. Pronađite event: "generate_lead"
3. Kliknite na toggle: "Mark as conversion" ✅
4. Sačuvajte

Sada možete pratiti koliko lead-ova dobijate!


───────────────────────────────────────────────────────────────────────────────
KORAK 3: Kreiranje Custom Report-a (Za detaljnu analizu)
───────────────────────────────────────────────────────────────────────────────

1. Idite na: Explore → Blank
2. Dodajte Dimensions:
   - Event name
   - Event label (za detalje)
   - Language (custom parameter)
   - Lead source (custom parameter)
3. Dodajte Metrics:
   - Event count (broj događaja)
   - Total users (broj korisnika)
4. Filter: Event name = "generate_lead" (ili drugi event)
5. Sačuvajte report


───────────────────────────────────────────────────────────────────────────────
KORAK 4: Gledanje Standardnih Report-ova
───────────────────────────────────────────────────────────────────────────────

Reports → Engagement → Events

Ovde možete videti:
- Koliko puta se svaki event desio
- Koliko korisnika je triggerovalo svaki event
- Trend po danima/nedeljama

NAJVAŽNIJE METRIKE ZA PRAĆENJE
═══════════════════════════════════════════════════════════════════════════════

1. ⭐ CONVERSION RATE (Lead generisanje)
   Formula: (generate_lead / page_views na /contact) × 100%
   Dobra vrednost: 5-10% za B2B

2. 📊 CTA EFFECTIVENESS (Koji CTA dugmad rade najbolje)
   Gde: Reports → Events → "cta_click" → event_label
   Pratite: Koji tekst/pozicija dugmeta privlači najviše klikova

3. 🎯 PORTFOLIO ENGAGEMENT (Koji projekti privlače pažnju)
   Gde: Reports → Events → "portfolio_click" → event_label
   Pratite: Koji projekti dobijaju najviše klikova

4. 🚫 FORM ABANDONMENT (Koliko ljudi počne ali ne završi formu)
   Formula: ((form_interaction / 3) - generate_lead) / (form_interaction / 3) × 100%
   Dobra vrednost: < 50%

5. 📈 LEAD SOURCE COMPARISON (Contact page vs Home page)
   Gde: Custom report sa "lead_source" dimension
   Pratite: Odakle dolaze lead-ovi

6. 🌍 LANGUAGE PERFORMANCE (Srpski vs Engleski korisnici)
   Gde: Custom report sa "language" dimension
   Pratite: Koji jezik bolje konvertuje
